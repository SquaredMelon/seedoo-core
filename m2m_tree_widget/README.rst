=================
M2M Tree Widget
=================

This module add a tree view to many2one field, for a model with a hierarchy

widget name=m2m_tree
options='{"all_checkable": "1","parent_field": "pid"}'

default values:
- all_checkable: False
- parent_field: parent_id
